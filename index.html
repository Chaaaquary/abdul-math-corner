Materi Utama

Pengertian Barisan & Deret

Barisan adalah kumpulan bilangan yang tersusun berurutan dengan pola tertentu. Setiap anggota disebut suku.

Deret adalah penjumlahan dari suku-suku suatu barisan bilangan.

Barisan:2, 5, 8, 11, 14, ...
Deret:2 + 5 + 8 + 11 + 14 + ...
Barisan Aritmetika

Pengertian & Rumus

Barisan dengan selisih (beda) tetap antar suku yang berurutan.

Rumus Suku ke-n
Uₙ = a + (n − 1)b
a = suku pertamab = bedan = nomor suku
Rumus Beda
b = Uₙ − Uₙ₋₁

Contoh Soal Barisan Aritmetika

Diketahui barisan: 3, 7, 11, 15, ...
Tentukan suku ke-20!
Pembahasan
a = 3,  b = 7 − 3 = 4,  n = 20
U₂₀ = 3 + (20 − 1) × 4
U₂₀ = 3 + 19 × 4 = 3 + 76
U₂₀ = 79
Deret Aritmetika

Rumus Jumlah Deret Aritmetika

Rumus Sₙ
Sₙ = n/2 × (2a + (n−1)b)
Rumus Alternatif
Sₙ = n/2 × (a + Uₙ)

Contoh Soal Deret Aritmetika

Tentukan jumlah 15 suku pertama dari barisan: 2, 5, 8, 11, ...
Pembahasan
a = 2,  b = 3,  n = 15
S₁₅ = 15/2 × (2×2 + 14×3) = 7.5 × 46
S₁₅ = 345
Barisan Geometri

Pengertian & Rumus

Barisan dengan rasio (perbandingan) tetap antar suku berurutan.

Rumus Suku ke-n
Uₙ = a × rⁿ⁻¹
a = suku pertamar = rasio
Rumus Rasio
r = Uₙ / Uₙ₋₁

Contoh Soal Barisan Geometri

Barisan: 2, 6, 18, 54, ...
Tentukan suku ke-7!
Pembahasan
a = 2,  r = 3,  n = 7
U₇ = 2 × 3⁶ = 2 × 729
U₇ = 1458
Deret Geometri

Rumus Jumlah Deret Geometri

Jika r ≠ 1
Sₙ = a(rⁿ − 1) / (r − 1)
Deret Tak Hingga (|r| < 1)
S∞ = a / (1 − r)

Contoh Soal Deret Geometri

Tentukan jumlah 6 suku pertama dari barisan: 3, 6, 12, 24, ...
Pembahasan
a = 3,  r = 2,  n = 6
S₆ = 3(2⁶ − 1)/(2−1) = 3 × 63
S₆ = 189
